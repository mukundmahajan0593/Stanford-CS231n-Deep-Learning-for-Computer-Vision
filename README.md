# Stanford-CS231n-Deep-Learning-for-Computer-Vision
“Reimplementation of key algorithms and models from Stanford’s CS231n course as part of independent study in computer vision.”

This work covers: Image Classification, kNN, SVM, Softmax, Fully Connected Neural Network
Original course assignment: https://cs231n.github.io/

## Dataset Setup
This project uses the CIFAR-10 dataset.
The dataset is **not included** in this repository.
It can be downloaded from:
https://www.cs.toronto.edu/~kriz/cifar.html
